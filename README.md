# MatchMind AI

> AI dating coach and profile optimizer

## Features
- Profile optimization
- Conversation starters
- Date ideas
- Compatibility insights
- Confidence coaching

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- OpenAI GPT-4o-mini
- Stripe Subscriptions
- Vercel deployment

## Quick Start

```bash
npm install
cp .env.example .env.local
# Fill in API keys
npm run dev
```
